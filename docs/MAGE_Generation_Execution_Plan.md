# MAGE 生成研究执行计划

## 2. 研究问题

RQ1. 可变掩码比例是否是实现强生成行为的必要条件？

RQ2. 生成行为中有多少来自 token 空间建模（离散语义 token），有多少来自采样策略？

RQ3. 在什么条件下，MAGE 能生成真正新颖的组合，而非从训练支撑集中近似检索/插值？

RQ4. 解码参数（`temp`、`num_iter`、置信度重掩码调度）如何控制保真度-多样性-坍缩之间的权衡？

## 3. 工作假设

H1. 可变掩码比例训练相比固定掩码比例能提升生成鲁棒性。

H2. 语义分词（VQ tokens）通过施加语义瓶颈，相比像素空间掩码目标能提升视觉一致性。

H3. 低温解码提升表观保真度，但增加模式坍缩和重复输出。

H4. "新颖"输出是条件依赖的，受后期解码步骤中去噪邻域重叠结构的约束。

## 4. 交付物

D1. 包含所有生成运行和配置的可复现实验矩阵。

D2. 包含 FID/IS、多样性/坍缩指标和定性网格图的结果包。

D3. 将每个结论明确映射到公式、代码路径和实验的理论-证据对照笔记。

D4. 最终结论页面："MAGE 何时能生成新样本，何时不能"。
